## Tips

1. **Save frequently** - Copy data to a text file after each town/season
2. **Check team counts** - Use expected counts above to verify you got all data
3. **Watch for missing seasons** - Some towns may not have data for older seasons
4. **Verify columns** - Make sure you're getting: Gender, Grade, Div Level, Div Tier, Team Name, W, L, T, Points, GF, GA, Goal Diff, Coach names
5. **Take breaks** - This is tedious work, don't burn out!
